# coupon-generator-core

## Description

![Screenshot of coupon](https://github.com/jprecigout/coupon-generator-core/blob/master/coupon.png)

Générateur de coupon de réduction

## Installation via docker

```bash
# generation des images
$ docker-compose build

# demarrage de la stack
$ docker-compose up
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
